# 32-bit ALU Design & Static Timing Analysis

## Overview
Designed and verified a 32-bit ALU in Verilog HDL supporting arithmetic and logical operations. Performed RTL synthesis and Static Timing Analysis (STA) using Yosys and OpenSTA with Sky130 standard-cell libraries.

## Features
- 32-bit Arithmetic Logic Unit
- Functional simulation and verification
- RTL synthesis using Yosys
- Gate-level netlist generation
- Static Timing Analysis using OpenSTA
- Sky130 standard-cell mapping

## Tools Used
- Verilog HDL
- Icarus Verilog
- GTKWave
- Yosys
- OpenSTA
- Sky130 Liberty Libraries

## Flow
RTL Design → Simulation → Synthesis → STA

## Timing Analysis
- WNS/TNS evaluation
- Timing constraint setup using SDC
- Gate-level timing analysis using OpenSTA
